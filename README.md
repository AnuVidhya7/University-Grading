University Exam Grading System (Java Console Application)
Project Overview
The University Exam Grading System is a console-based Java application designed to manage academic grading operations within a university environment. The system allows administrators to manage courses, students, assessments, record marks, publish grades, and generate student transcripts.
This project demonstrates the practical implementation of Object-Oriented Programming (OOP) concepts, collections framework, and menu-driven application design using Core Java.
Project Objectives
The system is designed to:
Add and manage courses
Register students
Create assessments for courses
Record student marks
Automatically calculate grades based on scores
Publish grades
Generate student transcripts
Features
Add Course
Add Student
Create Assessment
Record Marks
Automatic Grade Calculation (A+, A, B, C, D, F)
Publish Grades
